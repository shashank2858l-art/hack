# Student AI Assistant

An AI-powered academic helper with:
- Login/Signup
- Chat with Google Gemini (summaries, study plans, doubt solving)
- Upload files (PDF, images, text)
- Voice input + Camera capture
- All data saved per user

## Setup
1. `npm install`
2. Get Firebase config → replace in `src/js/firebase.js`
3. Get Gemini API key → https://aistudio.google.com/app/apikey → replace in `src/js/gemini.js`
4. `npm run dev` → opens at http://localhost:3000
